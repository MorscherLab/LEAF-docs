# Install in MINT

::: warning Under development
The LEAF plugin for MINT is not enabled for general use yet. Use the wheel + CLI install path for current analyses.
:::

LEAF's MINT deployment path is intended for future lab and shared-server use. When it is ready, MINT will handle authentication, file-folder access control, and multi-user isolation so the same LEAF instance can be reached from a browser on the lab network.

## Planned requirements

| | |
|---|---|
| **A running MINT instance** | Managed by the lab or server administrator. |
| **MINT admin role** | Plugin upload is admin-gated. |
| **The LEAF plugin bundle** | Not yet published for general use. |

## Current recommendation

- **[Install the wheel + CLI](/get-started/install-cli)** — current supported path for local single-user use on macOS / Windows / Linux.
- **[Desktop app](/get-started/install-desktop)** — native launcher, in development.
